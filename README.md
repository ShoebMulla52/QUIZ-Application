An simple Quiz Application that allows users to take a MCQ quiz and view the total score.
This app displays 4 options below one question the user can select any option among the 4 options.
And that can also help to give a simple test.

Features

1.Select correct option

2.Check score when attempting all questions

3.Easy to use and User-friendly design

4.Repeat options for Quiz

5.Accuricy of perfection

6.Use Local storage
